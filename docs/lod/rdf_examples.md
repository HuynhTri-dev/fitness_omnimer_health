# RDF Examples for Omnimer Health

This document provides example RDF (Turtle format) data generated by the `LODMapper` for various entities in the system. These examples serve as a reference for the Linked Data structure.

## Prefixes

```turtle
@prefix : <http://omnimer.health/data/> .
@prefix ont: <http://omnimer.health/ontology/> .
@prefix xsd: <http://www.w3.org/2001/XMLSchema#> .
@prefix sosa: <http://www.w3.org/ns/sosa/> .
@prefix ssn: <http://www.w3.org/ns/ssn/> .
@prefix snomed: <http://snomed.info/id/> .
@prefix loinc: <http://loinc.org/rdf/> .
@prefix fhir: <http://hl7.org/fhir/> .
@prefix schema: <http://schema.org/> .
@prefix foaf: <http://xmlns.com/foaf/0.1/> .
@prefix unit: <http://qudt.org/vocab/unit/> .
@prefix prov: <http://www.w3.org/ns/prov#> .
@prefix rdfs: <http://www.w3.org/2000/01/rdf-schema#> .
```

## 1. User Profile

Maps a user's basic information and consent status.

```turtle
:user_64f1b2c3e4b0d1a2f3c4e5d6 a schema:Person ;
    schema:gender schema:Male ;
    schema:birthDate "1995"^^xsd:gYear ;
    ont:hasConsent "true"^^xsd:boolean .
```

## 2. Health Profile

Represents a snapshot of health metrics (Weight, Height, BMI, etc.) as a SOSA Observation.

```turtle
:hp_64f1b2c3e4b0d1a2f3c4e5d7 a sosa:Observation ;
    sosa:hasFeatureOfInterest :user_64f1b2c3e4b0d1a2f3c4e5d6 ;
    sosa:resultTime "2023-10-27"^^xsd:date ;

    # BMI (LOINC 39156-5)
    sosa:hasMember [
        a sosa:Observation ;
        sosa:observedProperty loinc:39156-5 ;
        sosa:hasSimpleResult "22.5"^^xsd:decimal ;
        sosa:hasResultUnit unit:KiloGM-M2
    ] ;

    # Body Fat (LOINC 41982-0)
    sosa:hasMember [
        a sosa:Observation ;
        sosa:observedProperty loinc:41982-0 ;
        sosa:hasSimpleResult "15"^^xsd:decimal ;
        sosa:hasResultUnit unit:Percent
    ] ;

    # Resting Heart Rate (SNOMED 364075005)
    sosa:hasMember [
        a sosa:Observation ;
        sosa:observedProperty snomed:364075005 ;
        sosa:hasSimpleResult "60"^^xsd:integer ;
        sosa:hasResultUnit unit:BeatsPerMinute
    ] ;

    # Weight (LOINC 29463-7)
    sosa:hasMember [
        a sosa:Observation ;
        sosa:observedProperty loinc:29463-7 ;
        sosa:hasSimpleResult "70"^^xsd:decimal ;
        sosa:hasResultUnit unit:KiloGM
    ] ;

    # Height (LOINC 8302-2)
    sosa:hasMember [
        a sosa:Observation ;
        sosa:observedProperty loinc:8302-2 ;
        sosa:hasSimpleResult "175"^^xsd:decimal ;
        sosa:hasResultUnit unit:CentiM
    ] .
```

## 3. Watch Log (Daily Activity)

Represents daily activity data collected from a wearable device as a SOSA ObservationCollection.

```turtle
:wl_64f1b2c3e4b0d1a2f3c4e5d8 a sosa:ObservationCollection ;
    sosa:hasFeatureOfInterest :user_64f1b2c3e4b0d1a2f3c4e5d6 ;
    sosa:resultTime "2023-10-26"^^xsd:date ;

    # Device Information
    prov:wasGeneratedBy [
        a sosa:Sensor ;
        rdfs:label "Apple Watch" ;
        ont:deviceType "Series 8"
    ] ;

    # Steps (LOINC 55423-8)
    sosa:hasMember [
        a sosa:Observation ;
        sosa:observedProperty loinc:55423-8 ;
        sosa:hasSimpleResult "10500"^^xsd:integer
    ] ;

    # Calories Burned (LOINC 41981-2)
    sosa:hasMember [
        a sosa:Observation ;
        sosa:observedProperty loinc:41981-2 ;
        sosa:hasSimpleResult "450"^^xsd:decimal ;
        sosa:hasResultUnit unit:KiloCalorie
    ] ;

    # Sleep Duration (SNOMED 248263006)
    sosa:hasMember [
        a sosa:Observation ;
        sosa:observedProperty snomed:248263006 ;
        sosa:hasSimpleResult "420"^^xsd:integer ; # 7 hours
        sosa:hasResultUnit unit:Minute
    ] ;

    # Heart Rate Avg (SNOMED 364075005)
    sosa:hasMember [
        a sosa:Observation ;
        sosa:observedProperty snomed:364075005 ;
        sosa:hasSimpleResult "75"^^xsd:integer ;
        sosa:hasResultUnit unit:BeatsPerMinute
    ] .
```

## 4. Workout Session

Represents a completed workout session using `schema:ExerciseAction`.

```turtle
:wk_64f1b2c3e4b0d1a2f3c4e5d9 a schema:ExerciseAction ;
    schema:agent :user_64f1b2c3e4b0d1a2f3c4e5d6 ;
    schema:startTime "2023-10-27T08:00:00.000Z"^^xsd:dateTime ;
    schema:endTime "2023-10-27T09:00:00.000Z"^^xsd:dateTime ;

    # Summary Stats
    ont:totalCalories "300"^^xsd:integer ;
    ont:avgHeartRate "140"^^xsd:integer ;

    # Exercise Details
    schema:instrument [
        a ont:ExerciseSession ;
        ont:exerciseId "64f1b2c3e4b0d1a2f3c4e5e0" ; # Bench Press

        # Set 1
        ont:sets [
            a ont:Set ;
            ont:reps "10"^^xsd:integer ;
            ont:weight "60"^^xsd:decimal ;
            ont:order "1"^^xsd:integer
        ] ;

        # Set 2
        ont:sets [
            a ont:Set ;
            ont:reps "8"^^xsd:integer ;
            ont:weight "65"^^xsd:decimal ;
            ont:order "2"^^xsd:integer
        ]
    ] ;

    schema:instrument [
        a ont:ExerciseSession ;
        ont:exerciseId "64f1b2c3e4b0d1a2f3c4e5e1" ; # Squat

        # Set 1
        ont:sets [
            a ont:Set ;
            ont:reps "12"^^xsd:integer ;
            ont:weight "80"^^xsd:decimal ;
            ont:order "1"^^xsd:integer
        ]
    ] .
```

## 5. Goal

Represents a user's health or fitness goal using `fhir:Goal`.

```turtle
:gl_64f1b2c3e4b0d1a2f3c4e5da a fhir:Goal ;
    fhir:Goal.subject [
        fhir:Reference.reference "User/64f1b2c3e4b0d1a2f3c4e5d6"
    ] ;

    fhir:Goal.description [
        fhir:value "WeightLoss"
    ] ;

    fhir:Goal.startDate [
        fhir:value "2023-10-01"^^xsd:date
    ] ;

    # Target Metric
    fhir:Goal.target [
        fhir:Goal.target.measure [
            fhir:CodeableConcept.coding [
                fhir:Coding.system <http://loinc.org> ;
                fhir:Coding.code "Weight" # Ideally this would be a LOINC code
            ]
        ] ;
        fhir:Goal.target.detailQuantity [
            fhir:Quantity.value "65"^^xsd:decimal ;
            fhir:Quantity.unit "kg"
        ]
    ] .
```
